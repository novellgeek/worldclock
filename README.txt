Stylish World Clock App
=======================

Features:
- Add/remove cities live
- 24h or 12h time toggle
- Compact, Normal, and Large size modes
- Animated glow-style digital display
- Horizontal, readable layout with color-coded clocks

To Run:
-------
> python world_clock_stylish.py

To Build .EXE:
--------------
> pip install pyinstaller pytz
> pyinstaller --onefile --noconsole world_clock_stylish.py
